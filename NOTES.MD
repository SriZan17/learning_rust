Get documentation:
$rustup doc

To format code:
$cargo fmt

To attempt to auto fix code:
$cargo fix

Compile code:
$rustc main.rs

Create Project:
$cargo new project_name

Run built project:
$cargo run

Check if code compiles:
$cargo check

Building for release:
$cargo build release
